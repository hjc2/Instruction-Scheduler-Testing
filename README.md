## How to run

move the grap.sh file into your repo.

run the following command in your repo.

`$ ./grap.sh blocks`

that would run it on the blocks folder.

you could also do

`$ ./grap.sh blocks/report`

or

`$ ./grap.sh blocks/report/`

or

`$ ./grap.sh blocks/timing/`

really whatever
